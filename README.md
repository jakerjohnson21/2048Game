## Getting Started

Simply clone the repo and open the HTML file, or see it in action here: https://jakerjohnson21.github.io/2048Game/

## Instructions

Begin game with the 'New Game' button.

Then use keys to shift numbers:
  * 'W' : up
  * 'S' : down
  * 'A' : left
  * 'D' : right
